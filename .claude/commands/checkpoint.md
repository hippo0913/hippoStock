# /checkpoint

保存当前 session 进度。在每次对话结束前执行。

请完成以下操作：

1. 更新 PROGRESS.md：
   - "上次完成"改为本次 session 实际完成的内容
   - "下一步"改为下一个待执行的任务
   - "未解决的问题"记录任何遗留问题或待确认事项

2. 如果本次 session 产生了新的设计决策，追加到 DECISIONS.md，格式：
   ## D{编号}
   **决策**：
   **原因**：
   **影响**：

3. 如果完成了一个完整里程碑，追加到 CHANGELOG.md。

完成后输出一行摘要："进度已保存，下次从 [下一步任务] 继续。"
